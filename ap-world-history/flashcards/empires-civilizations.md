# Empires and Civilizations Quick Reference

Quick facts for the major empires and civilizations on the AP exam.

---

## Unit 1-2: 1200-1450

### Mongol Empire (1206-1368)
| Feature | Details |
|---------|---------|
| **Founder** | Genghis Khan |
| **Peak ruler** | Kublai Khan |
| **Region** | Central Asia, China, Persia, Russia |
| **Religion** | Tolerant of all religions |
| **Government** | Meritocracy; divided into khanates |
| **Economy** | Facilitated trade (Pax Mongolica) |
| **Military** | Cavalry; composite bow; psychological warfare |
| **Legacy** | Largest contiguous land empire; spread technology and disease |
| **Decline** | Fragmented into khanates; bubonic plague |

### Song Dynasty China (960-1279)
| Feature | Details |
|---------|---------|
| **Region** | China |
| **Religion** | Neo-Confucianism; Buddhism |
| **Government** | Merit-based bureaucracy; civil service exams |
| **Economy** | First paper money; urbanization; proto-industrialization |
| **Technology** | Gunpowder, compass, printing, Champa rice |
| **Legacy** | Economic revolution; technological innovation |
| **Decline** | Conquered by Mongols (Yuan Dynasty) |

### Yuan Dynasty (1271-1368)
| Feature | Details |
|---------|---------|
| **Founder** | Kublai Khan |
| **Region** | China |
| **Government** | Mongol rule over Chinese; foreigners favored |
| **Economy** | Continued trade; Marco Polo visited |
| **Legacy** | Connected China to wider Mongol trade network |
| **Decline** | Chinese resentment → Ming Dynasty |

### Ming Dynasty (1368-1644)
| Feature | Details |
|---------|---------|
| **Founder** | Zhu Yuanzhang (Hongwu Emperor) |
| **Region** | China |
| **Religion** | Neo-Confucianism |
| **Government** | Restored Chinese bureaucracy; civil service exams |
| **Economy** | Agricultural expansion; limited foreign trade |
| **Key events** | Zheng He voyages (1405-1433); Great Wall rebuilt |
| **Legacy** | Cultural flourishing; maritime withdrawal |
| **Decline** | Manchu invasion → Qing Dynasty |

### Delhi Sultanate (1206-1526)
| Feature | Details |
|---------|---------|
| **Region** | Northern India |
| **Religion** | Islam (ruling minority over Hindu majority) |
| **Government** | Sultanate; regional fragmentation |
| **Economy** | Agriculture; Indian Ocean trade |
| **Legacy** | Spread Islam to South Asia; Indo-Islamic culture |
| **Decline** | Weakened by invasions; replaced by Mughal Empire |

### Mali Empire (c. 1235-1600)
| Feature | Details |
|---------|---------|
| **Founder** | Sundiata Keita |
| **Peak ruler** | Mansa Musa |
| **Region** | West Africa |
| **Religion** | Islam (syncretic with traditional beliefs) |
| **Economy** | Trans-Saharan trade; gold and salt |
| **Key city** | Timbuktu (learning center) |
| **Legacy** | Wealth; spread of Islam in West Africa |
| **Decline** | Replaced by Songhai Empire |

### Aztec Empire (c. 1428-1521)
| Feature | Details |
|---------|---------|
| **Capital** | Tenochtitlan (on lake) |
| **Region** | Mesoamerica (central Mexico) |
| **Religion** | Polytheistic; human sacrifice |
| **Government** | Tribute empire; city-states |
| **Economy** | Tribute; chinampas (floating gardens) |
| **Military** | Flower wars; captured for sacrifice |
| **Legacy** | Large, complex civilization |
| **Decline** | Conquered by Cortés (1521) |

### Inca Empire (c. 1438-1533)
| Feature | Details |
|---------|---------|
| **Capital** | Cusco |
| **Region** | Andes (Peru, Ecuador, Chile, Bolivia) |
| **Religion** | Sun worship; emperor as divine |
| **Government** | Highly centralized; bureaucracy |
| **Economy** | Mit'a labor system; no money or markets |
| **Technology** | Roads; terrace farming; quipu |
| **Legacy** | Massive road system; administrative efficiency |
| **Decline** | Conquered by Pizarro (1533) |

## Unit 3-4: 1450-1750

### Ottoman Empire (1299-1922)
| Feature | Details |
|---------|---------|
| **Key rulers** | Mehmed II, Suleiman the Magnificent |
| **Capital** | Constantinople/Istanbul (after 1453) |
| **Region** | Middle East, North Africa, Southeast Europe |
| **Religion** | Sunni Islam; millet system for minorities |
| **Government** | Sultan; devshirme system; Janissaries |
| **Economy** | Trade crossroads; agriculture; tribute |
| **Military** | Gunpowder; Janissaries |
| **Legacy** | Long-lasting empire; bridge between East and West |
| **Decline** | "Sick Man of Europe"; WWI defeat |

### Safavid Empire (1501-1736)
| Feature | Details |
|---------|---------|
| **Founder** | Shah Ismail I |
| **Peak ruler** | Shah Abbas I |
| **Capital** | Isfahan |
| **Region** | Persia (modern Iran) |
| **Religion** | Shia Islam (state religion) |
| **Government** | Shah; theocratic |
| **Economy** | Silk trade; positioned between Ottomans and Mughals |
| **Military** | Qizilbash cavalry; later gunpowder |
| **Legacy** | Shia identity of Iran |
| **Decline** | Afghan invasion (1722) |

### Mughal Empire (1526-1857)
| Feature | Details |
|---------|---------|
| **Founder** | Babur |
| **Peak rulers** | Akbar, Shah Jahan |
| **Capital** | Delhi, Agra |
| **Region** | South Asia (India, Pakistan, Bangladesh) |
| **Religion** | Islam ruling Hindu majority; tolerance under Akbar |
| **Government** | Emperor; mansabdari system; zamindars |
| **Economy** | Cotton textiles; agriculture |
| **Architecture** | Taj Mahal |
| **Legacy** | Cultural flowering; Indo-Islamic architecture |
| **Decline** | Aurangzeb's intolerance; British takeover |

### Qing Dynasty (1644-1912)
| Feature | Details |
|---------|---------|
| **Founders** | Manchu people |
| **Key rulers** | Kangxi, Qianlong |
| **Region** | China + Tibet, Taiwan, Central Asia |
| **Religion** | Confucianism maintained |
| **Government** | Kept Ming bureaucracy; Manchu elite |
| **Economy** | Agriculture; trade surplus |
| **Control** | Queue required; Banner system |
| **Legacy** | Largest Qing territorial extent |
| **Decline** | Opium Wars; Taiping Rebellion; revolution (1912) |

### Tokugawa Japan (1603-1868)
| Feature | Details |
|---------|---------|
| **Founder** | Tokugawa Ieyasu |
| **Capital** | Edo (Tokyo) |
| **Region** | Japan |
| **Religion** | Buddhism; Confucianism; Shinto |
| **Government** | Shogunate; feudal; emperor as figurehead |
| **Social order** | Samurai, farmers, artisans, merchants |
| **Policy** | Sakoku (isolation) from West |
| **Legacy** | 250 years of peace; urban culture |
| **Decline** | Perry's arrival (1853); Meiji Restoration |

### Spanish Colonial Empire
| Feature | Details |
|---------|---------|
| **Region** | Americas (Mexico, Central/South America, Caribbean, Philippines) |
| **Religion** | Catholicism (conversion efforts) |
| **Government** | Viceroyalties; colonial bureaucracy |
| **Economy** | Silver mining; encomienda; haciendas |
| **Labor** | Encomienda; mita; African slavery |
| **Social** | Casta system (racial hierarchy) |
| **Legacy** | Destroyed indigenous empires; mixed cultures |
| **Decline** | Latin American independence (1810s-1820s) |

### Portuguese Colonial Empire
| Feature | Details |
|---------|---------|
| **Region** | Brazil; trading posts in Africa, India, Southeast Asia |
| **Government** | Factory system (trading posts, not territorial) |
| **Economy** | Sugar (Brazil); spice trade (Asia) |
| **Labor** | African slavery (especially Brazil) |
| **Legacy** | First global maritime empire |

## Unit 5-6: 1750-1900

### British Empire
| Feature | Details |
|---------|---------|
| **Peak** | 1920s (largest empire in history) |
| **Region** | Global: India, Africa, Americas, Australia, etc. |
| **Government** | Varied: colonies, protectorates, dominions |
| **Economy** | Industrial production; raw materials from colonies |
| **Key possession** | India ("Jewel in the Crown") |
| **Legacy** | English language; institutions; infrastructure |
| **Decline** | Decolonization after WWII |

### Russian Empire
| Feature | Details |
|---------|---------|
| **Key rulers** | Peter the Great, Catherine the Great, Alexander II |
| **Region** | Eastern Europe, Central Asia, Siberia |
| **Religion** | Russian Orthodox |
| **Government** | Absolute monarchy; serfdom until 1861 |
| **Expansion** | East to Pacific; south to Central Asia |
| **Economy** | Agriculture; industrialization lagged |
| **Legacy** | Multinational empire; Soviet successor |
| **Decline** | WWI; Russian Revolution (1917) |

### Meiji Japan (1868-1912)
| Feature | Details |
|---------|---------|
| **Government** | Emperor restored; constitution (1889) |
| **Policy** | Rapid Westernization/modernization |
| **Economy** | Industrialization; railroads; factories |
| **Military** | Modern army and navy |
| **Success** | Defeated China (1895), Russia (1905) |
| **Legacy** | Only non-Western nation to industrialize and become imperial power |

## Unit 7-9: 1900-Present

### Soviet Union (1922-1991)
| Feature | Details |
|---------|---------|
| **Founders** | Lenin, then Stalin |
| **Region** | Former Russian Empire |
| **Ideology** | Communism (Marxism-Leninism) |
| **Government** | One-party state; totalitarian under Stalin |
| **Economy** | Command economy; Five-Year Plans; collectivization |
| **Cold War** | Superpower rival to United States |
| **Legacy** | Alternative to capitalism; spread communism |
| **Decline** | Economic stagnation; Gorbachev reforms; 1991 collapse |

### Nazi Germany (1933-1945)
| Feature | Details |
|---------|---------|
| **Leader** | Adolf Hitler |
| **Ideology** | Fascism; Nazism; racism; anti-Semitism |
| **Government** | Totalitarian dictatorship |
| **Economy** | State-directed; rearmament |
| **Expansion** | Austria, Czechoslovakia, Poland → WWII |
| **Atrocity** | Holocaust (6 million Jews murdered) |
| **Decline** | Defeated in WWII (1945) |

### People's Republic of China (1949-present)
| Feature | Details |
|---------|---------|
| **Founder** | Mao Zedong |
| **Ideology** | Communism (Maoism); later market reforms |
| **Key events** | Great Leap Forward; Cultural Revolution |
| **Reforms** | Deng Xiaoping opens economy (1978) |
| **Economy** | Now world's second largest |
| **Legacy** | Communist party rule continues; economic rise |

## Comparison Charts

### Gunpowder Empires Compared
| Feature | Ottoman | Safavid | Mughal |
|---------|---------|---------|--------|
| Religion | Sunni Islam | Shia Islam | Islam (Hindu majority) |
| Peak ruler | Suleiman | Shah Abbas I | Akbar |
| Tolerance | Millet system | Shia conversion | Akbar tolerant |
| Elite troops | Janissaries | Qizilbash | Mansabdars |
| Ended | 1922 | 1736 | 1857 |

### Colonial Powers Compared
| Feature | Spanish | Portuguese | British | French |
|---------|---------|------------|---------|--------|
| Main region | Americas | Brazil, Asia | Global | Americas, Africa, Asia |
| Labor system | Encomienda | Slavery | Varied | Slavery, forced labor |
| Economic focus | Silver | Sugar, spices | Trade, industry | Trade, settlement |
| Religious mission | Strong | Strong | Moderate | Moderate |

### Responses to Western Pressure
| Country | Response | Result |
|---------|----------|--------|
| Japan | Meiji Restoration (modernize) | Became imperial power |
| China | Limited reforms | Humiliated; revolution |
| Ottoman | Tanzimat reforms | Continued decline |
| Egypt | Muhammad Ali's reforms | Partial success; British control |
